## Вектора
``` wolfram
v1 = {1, 2, 3}
v2 = {1, -2, 3}

Скалярное произведение делается через точку:
v1.v2
Либо так:
Dot[v1, v2]
{a1, b1, c1}.{a2, b2, c2} - выдаст a1a2 + b1b2 + c1c2

Векторное произведение:
Cross[v1, v2]
Либо v1 * v2

Длина вектора:
Norm[v1]

Угол между векторами:
VectorAngle[v1, v2]
```
## Матрицы
``` wolfram
a = {{1, 2}, {3, 4}}
b = {{-2, 2}, {3, 1}}

Перемножение матриц:
a.b

Отображение матрицы (картинка):
MatrixForm[a]

Возведение в степень:
MatrixPower[a, 4]

Определитель:
Det[a]

Обратная матрица:
Inverse[a] - если существует

Ранк матрицы
MatrixRank[a]

Транспонирование:
Transpose[a]

Единичная матрица:
IdentityMatrix[n], где n - количество строк и столбцов

Выделить строку матрицы:
a[[3]] - выдаст 3 строку

Выделить столбец:
c[[All, 2]] - выдаст 2 столбец

Выделить подматрицу
c[[2;;3, 2;;4]] - выдаст 2, 3 строки и 2, 3, 4 столбцы

Выделить диагональ:
Diagonal[a, n], где n - смещение вверх

Извлечение треугольных матриц:
UpperTriangularize[a] - верхнюю
LowerTriangularize[a] - нижнюю

Добавить столбец в конец:
d = Join[a, {{1}, {2}, {3}}, 2] - пишщем матрицу, потом каждый элемент столбца в фигурных скобках, а потом степень вложенности

Добавить строку:
Append[a, {1, 2, 3, 4}]

Приведение к каноническому виду:
RowReduce[a]

Решение СЛАУ:
LinearSolve[a, f], где f - список свободных коэффициентов
```
