#ЯПы 
## Определения
- **Поток** - минимальная единица выполнения кода
- **Мьютекс** - инструмент для блокировки ресурсов, предотвращающий одновременный доступ
## Примеры

Создание потока
```cpp
#include <thread>
void function() {};
std::thread myThread(function);
```

Использование мьютекса
```cpp
#include <mutex>
std::mutex mtx;
mtx.lock();
//доступ к общему ресурсу
mtx.
```
