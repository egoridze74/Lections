[[Автоматы]]

| (X, S) - входной символ и текущее состояние | $X_1$ | $\dots$ | $X_i$    | $\dots$ | $X_m$ |
| ------------------------------------------- | ----- | ------- | -------- | ------- | ----- |
| $S_1$                                       |       |         |          |         |       |
| $\dots$                                     |       |         |          |         |       |
| $S_j$                                       |       |         | $y_{ij}$ |         |       |
| $\dots$                                     |       |         |          |         |       |
| $S_n$                                       |       |         |          |         |       |
$y_{ij} = f(X_i, S_j)$

$A = (X, Y, S, f_y, f_s)$
$f_y$ - функция выходов символов ($f_y: \ S \times X \to Y$)
$f_s$ - функция переходов состояний ($f_s: \ S \times X \to S$)
