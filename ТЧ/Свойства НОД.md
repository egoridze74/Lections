1. НОД(a, b) = НОД(b, a)
2. НОД(a, $\varepsilon$) = 1, если $\varepsilon \in \mathbb{U}^*$
3. НОД(a, 0) = a
4. НОД(ac, bc) = c $\cdot$ НОД(a, b)
5. НОД(a, b) = НОД(a, b $\pm$ a)
6. НОД(a, b) = НОД(a, r), если $b = aq + r$
7. Пусть НОД(a, c) = 1 - взаимно-простые, тогда НОД(a, b $\cdot$ c) = НОД(a, b)
